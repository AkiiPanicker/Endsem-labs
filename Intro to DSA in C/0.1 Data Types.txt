
DATA TYPES :

integer -> int : Allows to take input as an integer                 Placeholder : %d
double -> decimal : allows to take input as decimal                 Placeholder : %f
long  - > decimals : allows to take input as decimal                Placeholder : %f
char -> character : allows to take input as a single letter/alphabet    Placeholder: %c
string -> Placeholder ": %s


variable naming rules:

1. cannot be a number ir symbol
2. alphabets or words can be variable names
3. symbols cannot be variable names
4. statements cannot be variables names ie printf, scanf, #include cannot be variable names
5. a = 45 , a1 = 56, a2= 78, a3 = 89, akshat67 = 217 ->allowed variable names
6. akshat_panicker = 987 , first_name = akshat, last_name = panicker ->allowed
